# Descriptions

Image file uploader with hyperapp + gRPC(Kotlin with Spring Boot) sample

## server-side

gRPC server side.

- Kotlin
- Spring Boot
- gRPC server

## client-side

- hyperapp

## start

```
cd ./message-front
npm i
```

```
cd ./k8s
kubectl apply deployment-mysql.yaml
```

```
 cd ./api
 ./gradlew bootRun
```
